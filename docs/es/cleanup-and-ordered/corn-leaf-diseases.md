# corn-leaf-diseases

## Common Rust (CR) | Puccinia sorghi
Imágenes validadas y procesadas. Se integraron bajo el identificador correspondiente tras superar los filtros de calidad.

## Gray Leaf Spot (GLS) | Cercospora zeae-maydis
Imágenes analizadas mediante PHash para descartar redundancias e integradas en el lote limpio.

## Northern Corn Leaf Blight (NCLB) | Exserohilum turcicum
Muestras incorporadas al set consolidado bajo condiciones de distribución estándar.

## Healthy | Sana
Imágenes de control integradas con éxito.

---

### Métricas de Extracción
* **Volumen extraído:** ~1,000 imágenes netas combinadas entre las clases principales.
* **Ubicación de destino:** `/data/clean/` distribuido según el entorno detectado (`real`/`lab`).
